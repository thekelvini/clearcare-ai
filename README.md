# ClearCare AI

ClearCare AI is a healthcare AI prototype that simplifies complex discharge instructions into patient-friendly language.

## Features
- AI-powered discharge simplification
- Nurse-reviewed workflow
- Plain-language patient education
- Local AI model using Ollama
- Hospital-style user interface

## Technologies
- Python
- Streamlit
- Ollama
- Llama 3.2
- Requests

## Clinical Safety
This tool does not diagnose, prescribe, or replace clinical judgment.
All outputs require healthcare professional review before patient use.
